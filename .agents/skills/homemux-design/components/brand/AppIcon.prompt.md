The HomeMux brand mark — a gable roof ("home") over a terminal block cursor — on the dark radial canvas with a mint gradient glyph.

```jsx
<AppIcon size={96} />          // full iOS tile
<AppIcon size={120} radius={26} />
<AppIcon bare size={28} />     // mint glyph only, for nav lockups
```

Use `bare` over dark backgrounds; the full tile already carries its own canvas.

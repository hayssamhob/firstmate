One-tap action button — filled mint for the primary action, tinted/bordered for secondary, plain for inline. Use `destructive` for Disconnect/Delete.

```jsx
<Button variant="prominent" size="large" fullWidth>Add First Host</Button>
<Button variant="bordered">Restore Purchase</Button>
<Button variant="plain" destructive>Close Session</Button>
```

Variants: `prominent` (filled, glows), `bordered` (translucent accent fill), `plain` (text). Sizes: `large`, `small`. Props: `destructive`, `fullWidth`, `icon`, `disabled`.

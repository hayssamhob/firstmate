Tappable icon — toolbar actions and the signature floating command button (a circular translucent material chip with the mint `command` glyph).

```jsx
<IconButton name="command" label="Command menu" shape="circle" size={52} />
<IconButton name="plus" label="Add host" variant="soft" />
<IconButton name="settings" label="Settings" variant="plain" tint="var(--text-secondary)" />
```

`variant`: material (blur chip), soft (tinted), plain. `shape`: rounded | circle.

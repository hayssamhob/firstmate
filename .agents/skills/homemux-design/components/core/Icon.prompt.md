Line icon primitive — wraps Lucide, our web substitute for SF Symbols. Inherits color via `currentColor`. Load the Lucide UMD script on the page first.

```jsx
<Icon name="terminal" size={22} color="var(--accent)" />
<Icon name="command" />
```

Common names used across HomeMux: `terminal`, `command`, `plus`, `settings`, `lock`, `key-round`, `mic`, `audio-lines`, `layers`, `rotate-cw`, `chevron-right`, `circle`. Stroke width 2 default.
